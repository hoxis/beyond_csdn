# 简介

建立本仓库的直接目的是为大家下载 CSDN 资源。

现在大多数人搜索资源时，排名靠前的都是 CSDN 的。

然而现在下载都是需要积分的，而大部分人都没有足够的积分。

由于本人手头有一些 CSDN 积分，所以一直在帮助网友进行下载。

后来打算做一个仓库，用来存放已下载的资源，避免重复下载。
